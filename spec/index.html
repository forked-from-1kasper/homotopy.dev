BINARY FORMAT

Published: 03 FEB 2022

Here is given a preliminary specification on Anders external term format.

CC is a byte.
QQ (from qword) is a 64-bit little-endian integer constisting of 4 bytes.
BB is a UTF-8 string encoded as its length followed by the string itself. In a BNF-like notation this written as B:=Qn C C … Cn timesB:=Qn C C … C⏟n times
ZZ represents here string (given by Z.to_bits function) that encodes Zarith integer.
I:=0016 | FF16 B QI:=0016 | FF16 B Q is a ident, where first part represents ignored variable (like in λa.λ_.aλa.λ_.a).
D:=0016 | FF16D:=0016 | FF16 represents interval elements (00 and 11 respectively).
F:=Qn (I D) … (I D)n timesF:=Qn (I D) … (I D)⏟n times is a face.
EE is a language expression.
S:=Qn (F E) … (F E)n timesS:=Qn (F E) … (F E)⏟n times is a cubical system.
